Petunjuk Penggunaan

Panduan fitur terbaru Web Timer Lomba v2.0

Memulai Penggunaan

1. Kontrol Timer di Kartu

  • Setiap timer memiliki 3 tombol utama:
    • Reset: Mengembalikan waktu ke posisi awal.
    • Pengaturan: Membuka panel konfigurasi di sisi kanan.
    • Mulai/Jeda: Menjalankan atau menghentikan timer sementara.

2. Mengatur Banyak Suara (Slot Suara)

  • Anda dapat menambahkan lebih dari satu suara untuk satu timer yang sama.
  • Klik "+ Tambah Suara Baru" untuk menambah slot suara tambahan.
  • Gunakan **Dropdown** untuk memilih file yang sudah ada di folder assets/.
  • Klik ikon **Play Hijau** () di setiap slot untuk mengetes bunyi suara sebelum disimpan.

3. Logika Trigger Suara PENTING

  • Mulai: Suara berbunyi sejak detik ke-0 (saat klik Play) dan akan berhenti otomatis pada detik ke-X yang diinput. Sangat cocok untuk musik intro.
  • Selesai: Suara mulai berbunyi saat sisa waktu mencapai X detik terakhir (countdown) dan berhenti saat timer habis.
  • Berputar: Suara akan terus berbunyi (loop) selama timer berjalan dari awal sampai akhir.

4. Mode Auto & Navigasi

  • Aktifkan Mode Auto di header agar timer berpindah otomatis ke tahap berikutnya (Persiapan → Lomba → Evaluasi).
  • Pada mode halaman terpisah, browser akan melakukan navigasi otomatis dan langsung menjalankan timer di halaman baru.

5. Mode Fokus (Tampilan Besar)

  • Jika Anda membuka halaman per-bagian (contoh: persiapan.html), tampilan timer akan menjadi **sangat besar** dan terpusat. Mode ini dirancang khusus untuk ditampilkan di proyektor agar terlihat oleh seluruh peserta lomba.
